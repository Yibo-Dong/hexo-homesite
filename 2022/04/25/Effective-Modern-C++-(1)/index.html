Item3 decltype

Usage:

  • help to compute return type:

  • implies not to stripe off the reference. Otherwise, rvalue cannot be assigned.

Strategy:

  • for lvalue expressions of type T, other than names, reports a type of T&.

Note:

  1. C++11 permits return types for single statement lambdas to be deduced, while C++14 extends to all lambdas and all functions.
  2. auto specifies that the type is to be deduced, and decltype says that decltype rules should be used.
  3. Putting parentheses around a name can change the type that decltype reports for it.
